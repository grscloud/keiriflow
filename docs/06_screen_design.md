# 06. 画面設計書（Screen Design）

---

# 1. 目的

本書では、KeiriFlowの画面構成および各画面のUI設計を定義する。

目的は以下の通り。

- UIの統一
- 開発者とデザイナー間の認識合わせ
- API設計との対応
- 将来的なSPA化への対応
